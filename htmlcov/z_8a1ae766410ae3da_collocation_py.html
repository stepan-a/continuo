1"""General collocation discretisation of the model residual. 

2 

3For the implicit DAE ``F(ẋ, x, e, θ, t) = 0`` an ``s``-stage collocation 

4method (:class:`~continuo.solve.disc.tableaux.ButcherTableau`) discretises 

5one interval ``[tᵢ, tᵢ₊₁]`` of step ``dt`` by introducing, per interval, 

6*stage unknowns*: the stage derivatives ``Vⱼ ∈ ℝ^{n_dyn}`` (the value of 

7``ẋ`` at the collocation point ``τⱼ = tᵢ + cⱼ·dt``) and the stage algebraic 

8values ``Wⱼ ∈ ℝ^{n_alg}``. The full stage state is 

9 

10 Zⱼ = [ xᵢ^dyn + dt·Σ_l A[j,l] V_l ; Wⱼ ] (endogenous order) 

11 

12and the per-interval residual stacks the ``s`` stage equations 

13 

14 F(Vⱼ, Zⱼ, e(τⱼ), θ, τⱼ) = 0 (s·n rows) 

15 

16— whose dynamic rows are the collocation conditions and whose algebraic rows 

17pin ``Wⱼ`` (index-1 DAE consistency at the stages) — followed by the dynamic 

18node update 

19 

20 xᵢ₊₁^dyn − xᵢ^dyn − dt·Σⱼ b[j] Vⱼ = 0 (n_dyn rows). 

21 

22The stage-derivative form never inverts ``A``, so it handles the singular 

23``A`` of Lobatto IIIA uniformly with Gauss and Radau, and at one Gauss stage 

24reduces to the Crank–Nicolson residual. The perfect-foresight driver stacks 

25these per-interval residuals across the grid (with the stage unknowns), keeps 

26the algebraic rows pointwise at the nodes too, and adds the boundary 

27conditions. 

28""" 

29 

30from __future__ import annotations 

31 

32import casadi as ca 

33 

34from continuo.codegen.residual import Residual 

35from continuo.solve.disc.tableaux import ButcherTableau 

36 

37__all__ = ["collocation_residual"] 

38 

39 

40def collocation_residual(residual: Residual, tableau: ButcherTableau) -> ca.Function: 

41 """Build the per-interval collocation residual for ``residual`` and ``tableau``. 

42 

43 The returned ``Function`` maps ``(x_i, x_next, V, W, e_stages, theta, 

44 t_i, dt)`` to the stacked residual of length ``s·n + n_dyn``: the ``s`` 

45 stage residuals (each the full ``F`` at a collocation point) followed by 

46 the dynamic node update. ``V`` / ``W`` / ``e_stages`` are the stage 

47 quantities concatenated stage-major. 

48 """ 

49 f = residual.function 

50 n = residual.expression.shape[0] 

51 n_dynamic = int(f.size_in("xdot")[0]) 

52 n_exogenous = int(f.size_in("e")[0]) 

53 n_parameters = int(f.size_in("theta")[0]) 

54 n_algebraic = n - n_dynamic 

55 s = tableau.stages 

56 A = tableau.A 

57 b = tableau.b 

58 c = tableau.c 

59 

60 x_i = ca.SX.sym("x_i", n) 

61 x_next = ca.SX.sym("x_next", n) 

62 V = ca.SX.sym("V", n_dynamic * s) 

63 W = ca.SX.sym("W", n_algebraic * s) 

64 e_stages = ca.SX.sym("e_stages", n_exogenous * s) 

65 theta = ca.SX.sym("theta", n_parameters) 

66 t_i = ca.SX.sym("t_i") 

67 dt = ca.SX.sym("dt") 

68 

69 def stage_v(j: int) -> ca.SX: 

70 return V[j * n_dynamic : (j + 1) * n_dynamic] 

71 

72 def stage_w(j: int) -> ca.SX: 

73 return W[j * n_algebraic : (j + 1) * n_algebraic] 

74 

75 def stage_e(j: int) -> ca.SX: 

76 return e_stages[j * n_exogenous : (j + 1) * n_exogenous] 

77 

78 x_i_dyn = x_i[:n_dynamic] 

79 stage_rows: list[ca.SX] = [] 

80 for j in range(s): 

81 z_dyn = x_i_dyn + dt * sum((A[j, m] * stage_v(m) for m in range(s)), ca.SX.zeros(n_dynamic)) 

82 z = z_dyn if n_algebraic == 0 else ca.vertcat(z_dyn, stage_w(j)) 

83 tau = t_i + c[j] * dt 

84 stage_rows.append(f(stage_v(j), z, stage_e(j), theta, tau)) 

85 

86 update = ( 

87 x_next[:n_dynamic] 

88 - x_i_dyn 

89 - dt * sum((b[j] * stage_v(j) for j in range(s)), ca.SX.zeros(n_dynamic)) 

90 ) 

91 

92 return ca.Function( 

93 "coll_interval", 

94 [x_i, x_next, V, W, e_stages, theta, t_i, dt], 

95 [ca.vertcat(*stage_rows, update)], 

96 ["x_i", "x_next", "V", "W", "e_stages", "theta", "t_i", "dt"], 

97 ["R"], 

98 )
